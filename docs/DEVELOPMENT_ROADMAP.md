# Development Roadmap

## Phase 1: Foundation

- Create Supabase project and apply schema.
- Implement signup, login, and profile hydration.
- Add guarded frontend routes.
- Complete CRUD endpoints for wardrobe and external products.

## Phase 2: Fit MVP

- Validate measurement forms by category.
- Add reference clothing selection.
- Run `POST /fit/recommend` from the frontend.
- Store fit analysis results and recommendation logs.

## Phase 3: Feedback Loop

- Add post-purchase feedback.
- Track clicked and purchased recommendations.
- Add basic analytics for recommendation accuracy.

## Phase 4: Automation

- Parse shopping mall size charts from pasted text.
- Add OCR flow for size chart images.
- Introduce brand and category calibration tables.

## Phase 5: Production Readiness

- Enable strict RLS policies.
- Add integration tests.
- Add deployment configs for Vercel, Railway, and Supabase.
- Add monitoring and structured logging.
